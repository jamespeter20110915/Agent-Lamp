# v1.0.0 Formal Release

## Scope

Formalize Agent-Lamp as a local status companion for Codex and Claude Code.

## Included Work

- Tab5 status display firmware with `idle`, `running`, `waiting`, `ok`, and `error` states.
- Wireless HTTP transport for daily use, with USB serial fallback for setup and recovery.
- Queue bridge daemon that receives hook status lines and forwards them to Tab5.
- Codex hook support through the queue bridge.
- Claude Code project-level hook support through `.claude/settings.json`.
- Completion sound support through `--done-sound` or `AGENT_LAMP_DONE_SOUND`.
- LaunchAgent configuration for a persistent background bridge.
- Tests for transport selection, hook mapping, queue/daemon state recovery, Claude Code config, and completion sound behavior.

## Validation

- `python3 -m unittest discover -s tests`
- `plutil -lint launchd/com.peterjames.agent-lamp.plist`
- Live daemon process confirmed with `--done-sound Glass`.
- Wireless `agent-lamp.local` ping verified outside the sandbox before release.

## Release Notes

This release is intended to be pushed to `main` and tagged as `v1.0.0`.
